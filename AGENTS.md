# Repository Instructions

This repository contains Laravel-focused AI skills and shared guidelines.

## Working Principles

- Keep each skill focused on one workflow, review area, or feature boundary.
- Prefer reusable guidance over repetition.
- Use the shared guidelines for architecture, validation, localization, frontend styling, Filament, Livewire, and review standards.
- Keep files small and easy to scan.
- Avoid duplicate or conflicting instructions across skills.

## Repository Layout

- `skills/` contains publishable skill folders.
- `guidelines/` contains shared guidance documents.

## Naming

- Skill folders should be descriptive and action-oriented.
- Guidelines should be short, topic-based, and reusable across projects.

## Maintenance

- Update shared guidelines before copying rules into many skills.
- Keep repository documentation concise and professional.
- Do not add empty folders or placeholder files.
