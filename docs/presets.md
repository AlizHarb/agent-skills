# Presets

Presets describe curated skill bundles for documentation, future automation, and agent discovery metadata.

Available presets:

- `core`
- `full`
- `filament`
- `api`
- `saas`
- `package`
- `security`
- `testing`
- `frontend`
- `devops`

The full skill library is shipped for Boost and skills.sh. Presets help developers and tools understand which skills are most relevant for a project type.
