A curated Laravel skills and guidelines repository for AI coding agents.

## What This Repository Contains

- `skills/` - publishable skill packages, each centered on a single workflow or review topic
- `guidelines/` - shared standards used across skills
- `.ai/` - local Boost source of truth for this workspace

## Design Goals

- Keep skills focused, reusable, and easy to copy into other Laravel projects
- Prefer version-aware guidance over generic advice
- Keep the repo organized so humans and agents can scan it quickly
- Avoid duplication by grouping shared policy in guidelines and keeping skills narrow

## Structure

The repository is organized into topic-based skill groups and supporting guidelines. See `skills/README.md` and `guidelines/README.md` for indexes.

## Publishing Notes

This repository mirrors the Laravel Skills layout so it can be published, copied, or adapted cleanly.
